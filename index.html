    Welcome to RxBotics

    It has only just begun but the dream is to create a fully functioning robotics controller framework using nothing but Javascript. This framework is initially going to be designed to work with Bonescript for the BeagleBone Black, but who knows what the future may hold!

    There is a wonderful course Control of Mobile Robots which has inspired me along this path, I recommend anyone interested to take a look. There is also instructions in that course on how to make a robot to use this framework with.

    This is early days, but watch this space!
